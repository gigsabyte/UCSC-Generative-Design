CMPM 147, Summer 2019

Assignment 3: Interactive NPC Artist with Generative Grammars

Controls: WASD/arrow keys to navigate, spacebar to interact/progress text, Y and N to answer prompts

An abstracted nondeterministic grammar is expanded into two different forms of 'art': a character generator and a musical haiku generator.

The character generator uses an axiom formatted with a markup language similar to HTML, with <tags> denoting a word to be replaced. The visual component of the character generator is based on matching the character's traits generated from its <tags>.

The musical haiku generator uses a much simpler format where each line of the axiom is either the number of syllables it should contain (for the haiku) or the type and length of its tune (for the music).
